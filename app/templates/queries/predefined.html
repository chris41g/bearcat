Query Types

  • All Online Hosts: Shows all hosts currently online
  • Search by IP/Subnet/Range: Find hosts using IP addresses, subnets, ranges, or wildcards
  • Hosts by OS: Filter hosts by operating system
  • Windows/Linux Hosts: Show hosts running specific OS types
  • Recent Scans: Show hosts scanned within a specified number of days

  • Hosts with Specific Port: Find hosts with a particular port open
  • Hosts with Network Shares: Show hosts with SMB/CIFS shares
  • Hosts with Admin Shares: Find hosts with C$, ADMIN$, or IPC$ shares

  • Hosts with Specific Software: Find hosts with particular software installed
  • Hosts with Foxit License: Show hosts with Foxit PDF license keys (original query)
  • Foxit Installs Without License: Find Foxit installations without license keys (likely subscriptions)
  • All Foxit Installs: Show all Foxit installations regardless of license status

  • Scan Session History: View all scan sessions with statistics
